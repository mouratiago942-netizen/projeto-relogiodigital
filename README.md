# projeto-relogiodigital
Esse é um projeto que desenvolvi para colocar em prática meus estudos de JavaScript , focado na manipulação do DOM e no uso de funções de tempo.
A ideia foi criar um relógio digital que se atualize em tempo real, com um visual limpo e direto ao ponto.

O que eu usei:
HTML5: Para estruturar a marcação dos números.

CSS3: Usei Flexbox para centralizar tudo na tela e dei um tapa no visual com um fundo darkcyane bordas arredondadas nos campos de hora.

JavaScript: Aqui é onde usei o setIntervalpara capturar a hora do sistema a cada segundo e uma função de formatação para garantir que os menores números que 10 sempre mostrem aquele "0" na frente (deixando o layout mais organizado).
